# UIU CSE Fest 2025 - Trackbots (Line Follower Robot)

## Overview
Participated in the **Line Follower Robot (LFR)** segment at **United International University (UIU) CSE Fest 2025** as part of the team **Pagla Ghora**. Achieving **5th place** out of 55 teams, this has been our best performance to date.

## Event Details
| Category | Details |
| --- | --- |
| Event | UIU CSE Fest 2025 |
| Segment | Line Follower Robot (LFR) |
| Organizer | United International University (UIU) |
| Venue | UIU Campus, Dhaka |
| Date | January 17, 2025 |
| Team | Pagla Ghora |
| Team Size | 5 members |
| Position | 5th Place |

## Competition Format
- Two rounds: preliminary and final.
- 55 teams participated in the competition.
- 18 teams qualified for the final round.

## Results
- Secured **5th Place** overall.
- Successfully competed in the finals among the top 18 qualifying teams.
- Significant milestone as our best result in a robotics competition so far.
